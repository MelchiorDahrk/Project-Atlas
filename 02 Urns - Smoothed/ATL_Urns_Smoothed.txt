Project Atlas: Urns Smoothed
Version: 1.0

Description
Replaces the meshes in the "Urns" set with completely new, atlased versions to improve performance (reducing drawcalls) in areas where these meshes are used.

Installation
Copy "meshes" folder to your Data Files directory
You will need the atlas texture from the original urns atlas mod, found on the project atlas download page: https://www.nexusmods.com/morrowind/mods/45399
